The `arc:on_get_hurt` action type is triggered when a player takes damage.

## Action Data

When this action is triggered, the following [action data](/docs/arc/wiki/action_data/index.mdx) is made available to be used by conditions and rewards:

<ActionDataTable actionType="get_hurt" />

## Example

The following example will give the player a diamond every time they take damage from a zombie.

```json title="data/my_datapack/arc/diamond_for_zombie_damage.json"
{
    "holder": {
        "type": "arc:player",
        "id": "arc:player"
    },
    "type": "arc:on_get_hurt",
    "conditions": [
        {
            "type": "arc:damage_source",
            "source": "mob",
            "direct_entity_type": "minecraft:zombie"
        }
    ],
    "rewards": [
        {
            "type": "arc:item",
            "item": {
                "id": "minecraft:diamond"
            }
        }
    ]
}
```
