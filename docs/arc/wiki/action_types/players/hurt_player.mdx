The `arc:on_hurt_player` action type is triggered when a player hurts another player.

## Action Data

When this action is triggered, the following [action data](/docs/arc/wiki/action_data/index.mdx) is made available to be used by conditions and rewards:

<ActionDataTable actionType="hurt_player" />

## Example

The following example will give the player a Strength I effect for 10 seconds every time they hurt another player.

```json title="data/my_datapack/arc/strength_on_hurt_player.json"
{
    "holder": {
        "type": "arc:player",
        "id": "arc:player"
    },
    "type": "arc:on_hurt_player",
    "conditions": [],
    "rewards": [
        {
            "type": "arc:effect",
            "effect": {
                "id": "minecraft:strength",
                "duration": 200,
                "amplifier": 0,
                "show_particles": false,
                "show_icon": false
            }
        }
    ]
}
```
