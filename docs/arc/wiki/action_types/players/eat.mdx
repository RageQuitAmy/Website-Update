The `arc:on_eat` action type is triggered when a player eats an item.

## Action Data

When this action is triggered, the following [action data](/docs/arc/wiki/action_data/index.mdx) is made available to be used by conditions and rewards:

<ActionDataTable actionType="eat" />

## Example

The following example will give the player a Hunger effect for 10 seconds when they eat a Golden Apple.

```json title="data/my_datapack/arc/hunger_on_eat.json"
{
    "holder": {
        "type": "arc:player",
        "id": "arc:player"
    },
    "type": "arc:on_eat",
    "conditions": [
        {
            "type": "arc:item",
            "item": {
                "id": "minecraft:golden_apple"
            }
        }
    ],
    "rewards": [
        {
            "type": "arc:effect",
            "effect": {
                "id": "minecraft:hunger",
                "duration": 200,
                "amplifier": 0,
                "show_particles": false,
                "show_icon": false
            }
        }
    ]
}
```
