The `arc:on_death` action type is triggered when the player dies.

## Action Data

When this action is triggered, the following [action data](/docs/arc/wiki/action_data/index.mdx) is made available to be used by conditions and rewards:

<ActionDataTable actionType="death" />

## Example

The following example will summon a skeleton holding a totem of undying at the player's location every time they die from fall damage.

```json title="data/my_datapack/arc/totem_for_fall_death.json"
{
    "holder": {
        "type": "arc:player",
        "id": "arc:player"
    },
    "type": "arc:on_death",
    "conditions": [
        {
            "type": "arc:damage_source",
            "source": "fall"
        }
    ],
    "rewards": [
        {
            "type": "arc:command",
            "command": "summon minecraft:skeleton %player_location% {equipment:{mainhand:{count:1,id:\"totem_of_undying\"}}}"
        }
    ]
}
```
