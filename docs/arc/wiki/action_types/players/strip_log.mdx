The `arc:on_strip_log` action type is triggered when a player strips a log or wood block using an axe.

## Action Data

When this action is triggered, the following [action data](/docs/arc/wiki/action_data/index.mdx) is made available to be used by conditions and rewards:

<ActionDataTable actionType="strip_log" />

## Example

The following example will give the player an oak log every time they strip a log or wood block.

```json title="data/my_datapack/arc/loot_on_strip_log.json"
{
    "holder": {
        "type": "arc:player",
        "id": "arc:player"
    },
    "type": "arc:on_strip_log",
    "conditions": [],
    "rewards": [
        {
            "type": "arc:item",
            "item": {
                "id": "minecraft:oak_log"
            }
        }
    ]
}
```
