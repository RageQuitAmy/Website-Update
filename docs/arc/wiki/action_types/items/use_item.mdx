The `arc:on_use_item` action type is triggered when a player uses an item (right-clicks with an item).

## Action Data

When this action is triggered, the following [action data](/docs/arc/wiki/action_data/index.mdx) is made available to be used by conditions and rewards:

<ActionDataTable actionType="use_item" />

## Example

The following example will give the player an emerald every time they use a carrot on a stick.

```json title="data/my_datapack/arc/use_carrot_for_emerald.json"
{
    "holder": {
        "type": "arc:player",
        "id": "arc:player"
    },
    "type": "arc:on_use_item",
    "conditions": [
        {
            "type": "arc:item",
            "item": {
                "id": "minecraft:carrot_on_a_stick"
            }
        }
    ],
    "rewards": [
        {
            "type": "arc:item",
            "item": {
                "id": "minecraft:emerald"
            }
        }
    ]
}
```
