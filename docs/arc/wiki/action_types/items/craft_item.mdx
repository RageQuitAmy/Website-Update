The `arc:on_craft_item` action type is triggered when a player crafts an item.

## Action Data

When this action is triggered, the following [action data](/docs/arc/wiki/action_data/index.mdx) is made available to be used by conditions and rewards:

<ActionDataTable actionType="craft_item" />

## Example

The following example will give the player a regeneration effect for 5 seconds every time they craft a diamond sword.

```json title="data/my_datapack/arc/regeneration_for_crafting_diamond_sword.json"
{
    "holder": {
        "type": "arc:player",
        "id": "arc:player"
    },
    "type": "arc:on_craft_item",
    "conditions": [
        {
            "type": "arc:item",
            "item": {
                "id": "minecraft:diamond_sword"
            }
        }
    ],
    "rewards": [
        {
            "type": "arc:effect",
            "effect": {
                "id": "minecraft:regeneration",
                "duration": 100,
                "amplifier": 0,
                "show_particles": false,
                "show_icon": false
            }
        }
    ]
}
```
