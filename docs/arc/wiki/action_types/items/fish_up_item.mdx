The `arc:on_fished_up_item` action type is triggered when a player fishes up an item.

## Action Data

When this action is triggered, the following [action data](/docs/arc/wiki/action_data/index.mdx) is made available to be used by conditions and rewards:

<ActionDataTable actionType="fish_up_item" />

## Example

The following example will give the player an emerald every time they fish up an item.

```json title="data/my_datapack/arc/fish_for_emerald.json"
{
    "holder": {
        "type": "arc:player",
        "id": "arc:player"
    },
    "type": "arc:on_fished_up_item",
    "conditions": [
        {
            "type": "arc:items",
            "items": ["#minecraft:fishes"]
        }
    ],
    "rewards": [
        {
            "type": "arc:item",
            "item": {
                "id": "minecraft:emerald"
            }
        }
    ]
}
```
