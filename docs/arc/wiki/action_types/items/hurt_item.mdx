The `arc:on_hurt_item` action type is triggered when a player hurts/damages an item.

## Action Data

When this action is triggered, the following [action data](/docs/arc/wiki/action_data/index.mdx) is made available to be used by conditions and rewards:

<ActionDataTable actionType="hurt_item" />

## Example

The following example

```json title="data/my_datapack/arc/no_hurt_pickaxes.json"
{
    "holder": {
        "type": "arc:player",
        "id": "arc:player"
    },
    "type": "arc:on_hurt_item",
    "conditions": [
        {
            "type": "arc:items",
            "items": ["#minecraft:pickaxes"]
        }
    ],
    "rewards": [
        {
            "type": "arc:cancel_action"
        }
    ]
}
```
