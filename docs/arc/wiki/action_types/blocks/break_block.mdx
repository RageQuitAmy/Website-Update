The `arc:on_break_block` action type is triggered when a player breaks a block.

## Action Data

When this action is triggered, the following [action data](/docs/arc/wiki/action_data/index.mdx) is made available to be used by conditions and rewards:

<ActionDataTable actionType="break_block" />

## Example

The following example will give the player a diamond every time they break a stone block.

```json title="data/my_datapack/arc/diamond_for_breaking_stone.json"
{
    "holder": {
        "type": "arc:player",
        "id": "arc:player"
    },
    "type": "arc:on_break_block",
    "conditions": [
        {
            "type": "arc:block",
            "block": "minecraft:stone"
        }
    ],
    "rewards": [
        {
            "type": "arc:item",
            "item": {
                "id": "minecraft:diamond"
            }
        }
    ]
}
```
