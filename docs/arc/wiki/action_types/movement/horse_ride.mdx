The `arc:on_horse_ride` action type is triggered when a player is riding a horse.

## Action Data

When this action is triggered, the following [action data](/docs/arc/wiki/action_data/index.mdx) is made available to be used by conditions and rewards:

<ActionDataTable actionType="horse_ride" />

## Example

The following example will give the player a Regeneration I effect for 10 seconds as long as they are riding a horse.

```json title="data/my_datapack/arc/regeneration_on_horse_ride.json"
{
    "holder": {
        "type": "arc:player",
        "id": "arc:player"
    },
    "type": "arc:on_horse_ride",
    "rewards": [
        {
            "type": "arc:effect",
            "effect": {
                "id": "minecraft:regeneration",
                "duration": 200,
                "amplifier": 0,
                "show_particles": false,
                "show_icon": false
            }
        }
    ]
}
```
