The `arc:on_crouch_stop` action type is triggered when a player stops crouching (sneaking).

## Action Data

When this action is triggered, the following [action data](/docs/arc/wiki/action_data/index.mdx) is made available to be used by conditions and rewards:

<ActionDataTable actionType="crouch_stop" />

## Example

The following example will give the player a Speed II effect for 10 seconds every time they stop crouching.

```json title="data/my_datapack/arc/swift_sneak_on_crouch.json"
{
    "holder": {
        "type": "arc:player",
        "id": "arc:player"
    },
    "type": "arc:on_crouch_stop",
    "rewards": [
        {
            "type": "arc:effect",
            "effect": {
                "id": "minecraft:speed",
                "duration": 200,
                "amplifier": 1,
                "show_particles": false,
                "show_icon": false
            }
        }
    ]
}
```
