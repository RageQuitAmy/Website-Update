The `arc:on_swim_start` action type is triggered when a player starts swimming.

## Action Data

When this action is triggered, the following [action data](/docs/arc/wiki/action_data/index.mdx) is made available to be used by conditions and rewards:

<ActionDataTable actionType="swim_start" />

## Example

The following example will give the player a Water Breathing effect for 10 seconds as long as they are swimming.

```json title="data/my_datapack/arc/water_breathing_on_swim_start.json"
{
    "holder": {
        "type": "arc:player",
        "id": "arc:player"
    },
    "type": "arc:on_swim_start",
    "rewards": [
        {
            "type": "arc:effect",
            "effect": {
                "id": "minecraft:water_breathing",
                "duration": 200,
                "amplifier": 0,
                "show_particles": false,
                "show_icon": false
            }
        }
    ]
}
```
