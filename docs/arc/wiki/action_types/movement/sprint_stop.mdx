The `arc:on_sprint_stop` action type is triggered when a player stops sprinting.

## Action Data

When this action is triggered, the following [action data](/docs/arc/wiki/action_data/index.mdx) is made available to be used by conditions and rewards:

<ActionDataTable actionType="sprint_stop" />

## Example

The following example will give the player a Slowness II effect for 10 seconds as long as they are not sprinting.

```json title="data/my_datapack/arc/slowness_on_sprint_stop.json"
{
    "holder": {
        "type": "arc:player",
        "id": "arc:player"
    },
    "type": "arc:on_sprint_stop",
    "rewards": [
        {
            "type": "arc:effect",
            "effect": {
                "id": "minecraft:slowness",
                "duration": 200,
                "amplifier": 1,
                "show_particles": false,
                "show_icon": false
            }
        }
    ]
}
```
